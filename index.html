Minimax

The Tic Tac Toe game implemented using the Minimax algorithm provides an intelligent and challenging opponent for players. The algorithm analyzes the current game state and recursively explores all possible moves and their outcomes, assigning a score to each possible move. It aims to minimize the maximum potential loss or maximize the minimum possible gain for the opponent.

The game unfolds as follows:

  1. Initialize the 3x3 game board.
  2. Prompt the first player to make a move.
  3. Update the game board with the player's move
  4. Check if the game is over (win, loss, or draw).
    • If so, display the result and end the game.
  5. If the game is not over, let the second player (AI) make a move using the Minimax algorithm.
    • The AI evaluates each available move by recursively simulating future moves and assigning scores.
    • It selects the move with the highest score if it is the AI's turn, or the move with the lowest score if it is the human's turn.
  6. Update the game board with the AI's move.
  7. Check if the game is over.
    • If so, display the result and end the game.
  8. If the game is not over, repeat steps 2-7 until the game reaches a win, loss, or draw state.
